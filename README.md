# Cinema Ticket Pricing Optimization using PSO

This project applies **Particle Swarm Optimization (PSO)** to optimize cinema ticket pricing.
The objective is to maximize revenue while maintaining affordable ticket prices.

## Features
- Single-objective optimization (Revenue maximization)
- Multi-objective optimization (Revenue vs Affordability)
- Interactive Streamlit dashboard
- Convergence visualization

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
